# 在超市结账

今天我去超市购物，终于到了结账的时候。售货员首先问我有没有会员卡，我差不多忘了，还好包里收拾得不错，很快找到了。

通过扫描会员卡，我可以打折，节省了不少钱！售货员把所有东西都收了，然后告诉我总共的收入……不对，是总价！我有点粗心，差点输错了密码。

我不仅买了蔬菜和水果，还尝了几种新零食。结账之后，我把东西装进袋子，顾客们都在排队，我不想打扰后面的人，所以动作很快。

一开始我还不太适应这里的自助结账机，但通过多次练习，现在已经完全掌握了。受不了排长队的感觉，所以我每次都选择自助！下次购物我会更有准备，不会再犯这种错误了。

## Grammar & Vocabulary

### Grammar Used

**[G1]**
- **Pattern** — 通过 (tōngguò) as preposition: method/means + purpose; also as verb meaning "to pass through"
- **Example 1** — 通过扫描会员卡，我可以打折。
*"By scanning my membership card, I can get a discount."*
- **Example 2** — 通过多次练习，现在已经完全掌握了。
*"Through repeated practice, I've now completely mastered it."*
- **Note** — Use 通过 + noun/verbal phrase at the start of a sentence to describe the method or means by which a result is achieved; it introduces the "how" before the outcome.

### Vocabulary Used

| Word / Phrase | Reading | English | Usage note |
|---|---|---|---|
| 购物 | gòu wù | shopping | Used to describe the general activity of shopping |
| 售货员 | shòu huò yuán | salesperson | The checkout staff member at the supermarket |
| 首先 | shǒu xiān | first of all | Introduces the first action in a sequence |
| 收拾 | shōu shi | to tidy up / to put in order | Here used to mean bag was well-organised |
| 收 | shōu | to collect / to take in | Seller scanning/collecting items at checkout |
| 打折 | dǎ zhé | to give a discount | Common in shopping contexts |
| 收入 | shōu rù | income / revenue | Used humorously — narrator confuses "总价" with "收入" |
| 粗心 | cū xīn | careless / thoughtless | Describes making a small mistake with the PIN |
| 输 | shū | to enter (a password) | Entering PIN at the payment terminal |
| 不仅 | bù jǐn | not only | Connects two purchasing actions |
| 尝 | cháng | to try / to taste | Tasting new snacks while shopping |
| 顾客 | gù kè | customer / client | Other shoppers queuing behind |
| 打扰 | dǎ rǎo | to disturb / to bother | Not wanting to hold up other customers |
| 动作 | dòng zuò | movement / action | Moving quickly to pack bags |
| 适应 | shì yìng | to adapt / to get used to | Getting used to the self-checkout machine |
| 受不了 | shòu bù liǎo | can't stand / unbearable | Can't stand long queues |
| 错误 | cuò wù | mistake / error | Vowing not to repeat mistakes next time |
