# 去看电影

李明的妻子叫小红。她很喜欢看电影。上个星期六，天气很晴，小红对李明说："我们去看电影吧！"

李明正在睡觉呢。小红叫他起床。"快起床！电影票只有两张，我们不能迟到！"

李明慢慢地起床，穿上衣服。他问："电影票多少钱？"小红说："每张票两千块。"李明说："两千块？！那不是错了吗？"小红笑着说："不贵！这是新电影，大家都想看！"

他们准备出发了。小红问："我们从哪里去电影院？坐火车还是走路？"李明说："我们坐火车吧，火车站就在近边。"

他们从火车站出发，到电影院。电影院在机场左边不远的地方。进电影院以前，小红买了一些东西吃——两个鸡蛋饼、一个大西瓜汁。她用铅笔在手机上画了一张地图，找到了他们的座位。

电影开始了。李明正在吃东西呢，小红说："不要吃了，认真看电影！"

李明看了一次，又问小红："这个电影是什么题目？"小红说："叫《红色的梦》。"李明说："为什么主角穿红色的衣服？"小红说："因为她是跳舞的演员！"

电影里，主角的丈夫去年离开了家。主角每天上班，很辛苦。她正在跳舞呢，突然接到一封信。信里说，她的丈夫在外国等她。她很高兴，希望快点见到他。

李明看了一个小时以后，真的哭了！小红笑他说："你平时打篮球、踢足球都很厉害，现在哭了！"大家都笑了。

电影完了以后，李明说："这个电影很好！我们下次还来吧。"小红说："好！介绍给姐姐也来看，她知道了一定喜欢。"

他们走出电影院，天已经晚上了。李明说："我们去外面吃饭吧？"小红说："好啊，但是我想先休息一小时，我们慢慢走去找餐厅。"

他们往火车站方向走，找了一家好餐厅。李明说："今天很开心！"小红说："是啊，因为天晴，票也不贵，电影也很好看！"他们两个人都很高兴，在晚上的街上走着，说说笑笑，非常幸福。

## Grammar & Vocabulary

### Grammar Used

- **Pattern** — [G1] Progressive Tense: 正在……呢 / 在……呢 / 正……呢
- **Example** — 李明正在睡觉呢。(*Lǐ Míng zhèngzài shuìjiào ne.* — "Li Ming is in the middle of sleeping.")；小红正在跳舞呢 (*Xiǎo Hóng zhèngzài tiàowǔ ne* — "Xiao Hong is dancing right now.")；李明正在吃东西呢 (*Lǐ Míng zhèngzài chī dōngxi ne* — "Li Ming is eating right now.")
- **Note** — Use 正在……呢 to emphasise that an action is currently happening at this very moment; 正在 and 呢 can be used together or separately for the same progressive meaning.
